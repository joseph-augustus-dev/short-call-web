# Board Documentation Pack

This folder contains management-facing collateral for the Short Call Web initiative.

## Immediate local deliverables

- `management-proposal.html` — board-ready proposal document (open locally, print to PDF).
- `board-presentation.html` — local presentation deck in full-screen slide format.
- `local-deliverables.md` — quick local usage guide.

## Editable source files

- `management-proposal.md` — formal proposal source.
- `board-presentation.md` — Marp-compatible slide source.

## Use locally now

```bash
python3 -m http.server 4173
```

Open:

- `http://localhost:4173/docs/board/management-proposal.html`
- `http://localhost:4173/docs/board/board-presentation.html`

Print either page to PDF from your browser.

## Optional export tools

If you prefer generated binary files from markdown:

- Proposal PDF via `pandoc`
- Deck PPTX/PDF via `marp`

(see command examples in previous revisions if those tools are installed.)
